## Catatan Perjalanan

Mencatat Perjalanan Anda

## Instalasi
1. git clone https://github.com/agayanu/catatan-perjalanan.git
2. cd catatan-perjalanan
3. composer install
4. rename ".env.example" to ".env"
5. php artisan key:generate
